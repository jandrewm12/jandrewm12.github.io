Course Description
INFOTC 4320 - Software Engineering Software Engineering covers the principles, processes, and professional practices used to design, develop, test, deploy, and manage software systems in a team-based, collaborative environment. A range of software engineering methodologies are covered with an emphasis on agile software development using incremental methods of managing the development activities.
INFOTC 3850 - Computer System Administration This course is an introduction to Computer System Administration, we introduce the knowledge, skills, and procedures required for the stable and reliable deployment, management, and administration of an organization's computer systems. The course covers the planning, design, installation, maintenance, security, management tasks, tools, responsibilities, and ethics associated with being a Computer System Administrator for Unix/Linux server and MS Windows servers.
INFOTC 3380 - Database Systems and Applications This course covers database management systems (DBMS) and the development of applications that utilize databases including relational/SQL and NoSQL types. Topics include the evolution of data storage and databases, data modeling, relational and NoSQL databases, SQL, document, graph and key-value storage and retrieval, application development using databases, database scaling, database trends, and popular database systems.
INFOTC 3330 - Object Oriented Programming This course focuses on object-oriented programming concepts such as: Abstraction, Polymorphism, Encapsulation, Inheritance, Interfaces, Abstract Classes, Files, Streams, and Object Serialization. Topics such as GUI and event-driven programming, APIs, and design patterns are also tackled.
INFOTC 2830 - Web Application Development 1 This course will attempt to provide a comprehensive understanding of the evolution, the technologies, and the tools of the Internet. In particular, issues pertaining to the World Wide Web and Multimedia (HTML, CGI, Web based applications) will be discussed in detail.
